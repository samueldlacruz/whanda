Documentation

Complete API reference. Tags: [core] essential, [conv] convenience, [plugin] requires plugin.

Installation

No npm required. Just copy the files from dist/ to your project.

whanda.min.js ~26KB whanda-plugins.min.js ~4KB whanda.esm.min.js ~31KB 
<!-- Core (~26KB) -->
<script src="whanda.min.js"></script>

<!-- Optional plugins (~4KB) -->
<script src="whanda-plugins.min.js"></script>

ES Module

import { Whanda } from "./whanda.esm.min.js";
import { initPlugins } from "./whanda.esm.min.js";

initPlugins(Whanda);

Quick start

const w = new Whanda({ whatsappNumber: "1234567890" });

w.setProducts([
  { id: "1", name: "T-Shirt", price: 1200, stock: 25, category: "Clothing", image: "tshirt.jpg" }
]);

await w.addItem("1");
w.setCustomerName("Juan");
w.setCustomerAddress("123 Main St");
w.setPaymentMethod("Cash");
w.setDeliveryMethod("Home Delivery");

const order = await w.createOrder();
const link = await w.sendToWhatsApp(order);
window.open(link, "_blank");

Quick start with AI

Generate a complete store by pasting a prompt into ChatGPT, Claude, Cursor, or any AI agent.

Step 1: Download the files above (whanda.min.js + whanda-plugins.min.js)

Step 2: Copy the prompt below and paste it into your AI agent

Using whanda.min.js and whanda-plugins.min.js,
create a [TYPE] store in a single HTML file.

IMPORTANT:
- No npm. No import statements.
- Copy whanda.min.js and whanda-plugins.min.js
  to the same directory as the HTML.

Flow: Catalog → Cart → WhatsApp → Manual close

Requirements:
1. <script src="whanda.min.js">
2. new Whanda({ whatsappNumber: "NUM" })
3. w.setProducts([...])
4. await w.addItem(id)
5. w.setCustomerName() / setCustomerAddress()
6. const order = await w.createOrder()
7. window.open(await w.sendToWhatsApp(order))
8. Responsive design with Tailwind CSS

Hooks

Use w.on("hook", fn) to register and w.off("hook", fn) to unsubscribe.

HookPayloadFires when
beforeAddToCart{ productId, quantity }Before adding item
afterAddToCartcart itemsAfter adding item
afterCartChangecart itemsAfter any cart change
beforeCheckoutmetaBefore creating order
afterCheckoutorderAfter creating order
beforeWhatsAppSendorderBefore generating link
onRemoveItem{ productId, cart }Removing item from cart
onCartEmpty{ cart }Emptying the cart

Hooks can abort by returning { abort: true, message: "..." }
