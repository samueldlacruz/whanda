Instalación

No requiere npm. Solo copia los archivos desde dist/ a tu proyecto.

whanda.min.js ~26KB whanda-plugins.min.js ~4KB whanda.esm.min.js ~31KB 
<!-- Core (~26KB) -->
<script src="whanda.min.js"></script>

<!-- Plugins opcionales (~4KB) -->
<script src="whanda-plugins.min.js"></script>

ES Module

import { Whanda } from "./whanda.esm.min.js";
import { initPlugins } from "./whanda.esm.min.js";

initPlugins(Whanda);

Empezar rápido

const w = new Whanda({ whatsappNumber: "1234567890" });

w.setProducts([
  { id: "1", name: "Camiseta", price: 1200, stock: 25, category: "Ropa", image: "camiseta.jpg" }
]);

await w.addItem("1");
w.setCustomerName("Juan");
w.setCustomerAddress("Calle Principal 123");
w.setPaymentMethod("Efectivo");
w.setDeliveryMethod("Entrega a domicilio");

const order = await w.createOrder();
const link = await w.sendToWhatsApp(order);
window.open(link, "_blank");

Empezar rápido con AI

Genera una tienda completa pegando un prompt en ChatGPT, Claude, Cursor o cualquier agente de AI.

Paso 1: Descarga los archivos anteriores (whanda.min.js + whanda-plugins.min.js)

Paso 2: Copia el prompt de abajo y pégalo en tu agente de AI

Usando whanda.min.js y whanda-plugins.min.js,
crea una tienda de [TIPO] en un solo archivo HTML.

IMPORTANTE:
- Sin npm. Sin declaraciones import.
- Copia whanda.min.js y whanda-plugins.min.js
  al mismo directorio que el HTML.

Flujo: Catálogo → Carrito → WhatsApp → Cierre manual

Requisitos:
1. <script src="whanda.min.js">
2. new Whanda({ whatsappNumber: "NUM" })
3. w.setProducts([...])
4. await w.addItem(id)
5. w.setCustomerName() / setCustomerAddress()
6. const order = await w.createOrder()
7. window.open(await w.sendToWhatsApp(order))
8. Diseño responsivo con Tailwind CSS

Hooks

Usa w.on("hook", fn) para registrar y w.off("hook", fn) para cancelar la suscripción.

HookDatosSe ejecuta cuando
beforeAddToCart{ productId, quantity }Antes de agregar artículo
afterAddToCartcart itemsDespués de agregar artículo
afterCartChangecart itemsDespués de cualquier cambio en el carrito
beforeCheckoutmetaAntes de crear pedido
afterCheckoutorderDespués de crear pedido
beforeWhatsAppSendorderAntes de generar enlace
onRemoveItem{ productId, cart }Al eliminar artículo del carrito
onCartEmpty{ cart }Al vaciar el carrito

Los hooks pueden abortar retornando { abort: true, message: "..." }
