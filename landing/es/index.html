Cómo funciona

De descubrimiento a WhatsApp en 5 pasos

1

Descubre

Tu cliente encuentra tus productos a través de redes sociales, anuncios o recomendaciones.

2

Explora

Explora tu catálogo — busca, filtra, ve precios y stock en tiempo real.

3

Carrito

Agrega artículos, aplica cupones, elige envío y pago. Todo validado.

WhatsApp

Recibes los detalles completos del pedido — artículos, precio, dirección — por WhatsApp.

5

Cierra

Confirmas, arreglas el pago y la entrega — manualmente, a tu manera.

Características

Todo lo que necesitas,
nada que no

Carrito de Compras

Validación de stock en tiempo real, cantidades, persistencia con localStorage y cache TTL.

Precios Dinámicos

Envío fijo, por artículo o gratis por encima de un umbral. Multi-monedas con conversión automática.

Cupones

Monto fijo, porcentaje, expiración, límites de uso, pedido mínimo. Control total.

WhatsApp

Generación de enlaces, compartición de catálogo, página de agradecimiento. Plantillas personalizables.

Multi-Región

Impuestos y envío por región. ITBIS en RD, IVA en otros. Conversión automática.

Estado del Pedido

Pendiente, confirmado, enviado, entregado, cancelado. Seguimiento completo de pedidos.

Sistema de Hooks

Personaliza cualquier paso del flujo. beforeAddToCart, afterCheckout, beforeWhatsAppSend.

Seguridad

Sanitización de strings, validación JSON, rate limiting. Protección XSS incluida.

Inicio rápido

Listo en 10 líneas

// 1. Crear instancia
const w = new Whanda({ whatsappNumber: "1234567890" });

// 2. Cargar productos
w.setProducts([
  { id: "1", name: "T-Shirt", price: 1200, stock: 25, category: "Clothing", image: "tshirt.jpg" }
]);

// 3. Agregar al carrito
await w.addItem("1");

// 4. Información del cliente
w.setCustomerName("Juan");
w.setCustomerAddress("123 Main St");
w.setPaymentMethod("Cash");
w.setDeliveryMethod("Home Delivery");

// 5. Crear pedido y enviar a WhatsApp
const order = await w.createOrder();
const link = await w.sendToWhatsApp(order);
window.open(link, "_blank");

Google Sheets como tu base de datos

Carga productos directamente desde un Google Sheet. El adaptador convierte CSV a productos Whanda. Configura tu hoja para que "Cualquier persona con el enlace pueda ver."

  • Sin servidor, sin base de datos, sin configuración
  • Columnas requeridas: id, name, price, stock, category, image
  • Proxy serverless para producción (Vercel, Netlify) 
const w = new Whanda({
  whatsappNumber: "1234567890"
});

// Cargar desde Google Sheets
await loadFromSheets(w, {
  sheetUrl: "https://docs.google.com/
    spreadsheets/d/SHEET_ID/
    edit?gid=0"
});

// Productos listos
w.getProducts();
// → [{ id, name, price, ... }]

Vibecoding

Genera tu tienda con AI

Copia este prompt y pégalo en tu agente de IA favorito para generar una tienda completa en segundos. 

Usando whanda.min.js y whanda-plugins.min.js,
crea una tienda de [TIPO] en un solo archivo HTML.

IMPORTANTE:
- Sin npm. Sin statements de import.
- Copia whanda.min.js y whanda-plugins.min.js
  al mismo directorio que el HTML.

Flujo: Catálogo → Carrito → WhatsApp → Cierre manual

Requisitos:
1. <script src="whanda.min.js">
2. new Whanda({ whatsappNumber: "NUM" })
3. w.setProducts([...])
4. await w.addItem(id)
5. w.setCustomerName() / setCustomerAddress()
6. const order = await w.createOrder()
7. window.open(await w.sendToWhatsApp(order))
8. Diseño responsivo con Tailwind CSS
